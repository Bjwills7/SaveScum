Select the save file you want to use. It should start with Exanima, end with a number, and have a file extension of ".rsg". The folder that contains it should be at c:\Users\"your username"\AppData\Roaming\Exanima. Everytime your game is saved this tool replaces your checkpoint with the main save, if you want to manually save simply exit to the menu and the game will save.

YOU SHOULD BACKUP THE SAVE FOLDER MANUALLY BEFORE USING THIS TOOL.
